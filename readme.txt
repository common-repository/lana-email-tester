== Description ==

Tester plugin to try WordPress email sending function.

= Lana Codes =
[Lana Email Tester](https://lana.codes/product/lana-email-tester/)

== Installation ==

= Requires =
* WordPress at least 4.0
* PHP at least 5.3

= Instalation steps =

1. Upload the plugin files to the `/wp-content/plugins/lana-email-tester` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

= How to use it =
* in `Tools > Lana Email Tester`, you can send test email to the email address you provided.
